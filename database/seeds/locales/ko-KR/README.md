# ko-KR Seed Locale (Reserved)

This directory is a reserved placeholder for Korean (South Korea) seed data.

Per `DATABASE_FRAMEWORK_SPEC.md` section 8.1, reserved locales must keep directory
placeholders so future activation does not require structural migration.

Activation: add `"ko-KR"` to `activeLocales` in `seed.manifest.json` and populate
seed SQL files here when Korean locale initialization data is needed.
